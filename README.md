### repository 의 목적

- 공부 내용 저장소
- 프로젝트를 포함하여 문제 발생 시 해결 과정 기록 (일종의 오답 노트)
- 연습 코드 업로드
- 책, 강의(유튜브, 학교) 요약 및 정리
- 참고할 만한 유용한 사이트 정리
- 책 소화하기: 요약, 빨간펜만 따로, 키워드 정리

### Commit Convention

- Feat: 새로운 파일 추가
- Update: 코드 수정
- Remove : 코드 삭제
- Chore: 기타
